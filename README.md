Distributed system for cracking MD5 hashes

### Requirements

- python 3.10
- Docker version 26.1.3
- docker-compose version 1.29.2
  
### Запуск
```sudo docker-compose down && sudo docker-compose build --no-cache && sudo docker-compose up```
